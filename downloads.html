Latest release

It has been decided not to distribute release files (i.e. archives containing a bundle of code, scripts, and documentation files) any more. This is to reduce maintenance efforts and to prevent the download of partly outdated material. In order to get (or upgrade to) the software's latest version, please clone the ECHSE's code repository (see link in left panel of the main page). Don't forget to contribute your own developments being of potential interest to other users.
